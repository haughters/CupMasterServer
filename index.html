CupMaster Server

A Java RESTful WebService to choose a McDonald's coffee.
Java Spring Boot Heroku

About

This project acts as the webservice component of an app that chooses which coffee from McDonalds to choose. It is based upon a Spring Boot web application and accepts REST requests to perform operations.

Current Functionality

  • Display all configured coffees
  • Randomly choose a coffee for the user based on preference
  • Set preferences for each individual coffee

Desired Functionailty

  • Data Persistence
  • Swagger Documentation
  • Add Coffees via Post
